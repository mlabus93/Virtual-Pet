Thank you for downloading Fluvio! 
To install Fluvio, go to Window > Fluvio > Install Fluvio. To get started, go to Window > Fluvio > Welcome.

The reference manual for this version can be found by going to Window > Fluvio > Reference Manual. This manual can also be downloaded locally by going to the Fluvio preference pane in the Unity preferences.

The up-to-date reference manual is available at:
https://docs.thinksquirrel.com/fluvio/reference/

Product support can be found at:
https://support.thinksquirrel.com

For all example scenes and additional Fluvio packages, please explore Fluvio on the Asset Store:
https://www.assetstore.unity3d.com/#!/search/fluvio

For information on other Thinksquirrel extensions, check out:
